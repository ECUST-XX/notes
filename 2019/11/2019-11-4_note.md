### 内存泄漏是指由于疏忽或错误造成程序未能释放已经不再使用的内存.

```php
<?php
$a = [1];
// 自己引用自己导致内存泄漏
$a[1] = &$a;
var_dump($a);
// 输出类似如下:
// array(2) {
//   [0]=>
//   int(1)
//   [1]=>
//   &array(2) {
//     [0]=>
//     int(1)
//     [1]=>
//     *RECURSION*
//   }
// }
```
